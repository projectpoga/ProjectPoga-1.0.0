####################################
#Instructions to set up the project#
####################################

1.Clone the project talentvouch-1.0.0
2.Find TV_DATA.sql script under the directory talentvouch\database\ddl\create_table and execute the script
3.Modify the talentvouch.properties file under the directory talentvouch\properties according to the data base configuration
4.Set log path in log4j2.xml in codebase\talentvouch\src\main\resources to the corresponding directory where talentvouch.log exists
5.Run the project talentvouch
